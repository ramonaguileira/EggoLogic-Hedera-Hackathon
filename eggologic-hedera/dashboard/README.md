# Eggologic Dashboard

React + Tailwind dashboard with HashPack wallet integration.

## Features

- **PointsBalance**: Shows supplier's EGGOCOINS balance
- **DeliveryTable**: Recent deliveries with quality grades
- **CarbonTracker**: Progress bar toward next CARBONCOIN NFT
- **TrustchainViewer**: Verify any delivery's on-chain evidence
- **WalletConnect**: HashPack/Blade wallet integration

## Setup

```bash
npm install
npm run dev  # Opens at http://localhost:5173
```
