# Smart Contracts (Not Used)

Eggologic uses **Hedera Token Service (HTS)** for tokens and **Hedera Consensus Service (HCS)** for audit trails — both managed through Guardian's policy engine. No smart contracts are needed.

| Service | Purpose | How |
|---|---|---|
| **HTS** (Fungible) | EGGOCOIN (`0.0.8287358`) | Minted by Guardian when VVB approves Waste Delivery |
| **HTS** (NFT) | CIN (`0.0.8287362`) | Minted by Guardian when VVB approves Impact Calculation |
| **HCS** | Policy topic (`0.0.8291451`) | All VCs and policy messages recorded here |

This directory is reserved for future smart contract needs (e.g., automated escrow for CIN NFT marketplace).
